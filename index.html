Bridging Data and Action

This dashboard represents a full-stack approach to resolving administrative bottlenecks in educational institutions. By transitioning from isolated Excel spreadsheets to a centralized, relational SQLite database powered by a Node.js API, we transformed raw attendance logs into live, actionable intelligence.

The resulting system not only automates the calculation of critical KPIs—such as identifying at-risk cohorts falling below the 75% threshold—but also provides beautifully rendered, interactive visualizations allowing stakeholders to make data-driven decisions instantly.

The Challenge

Manual Tracking Overhead

Traditional attendance tracking relies on disparate Excel sheets, leading to fragmented data, human error, and significant administrative overhead without generating actionable insights.

Delayed Risk Identification

Without automated thresholds, identifying at-risk students (falling below 75% attendance) happens too late in the academic cycle for meaningful intervention.

Lack of Structured Reporting

Stakeholders lack a unified, high-level view of attendance health across subjects and timeframes, making data-driven academic policy decisions difficult.

preprocessing.py
# Load raw attendance log
df = pd.read_excel('raw_attendance.xlsx')

# Remove duplicate entries
df.drop_duplicates(subset=['Student_ID', 'Date'], inplace=True)

# Handle missing identifiers
df.dropna(subset=['Student_ID'], inplace=True)

# Standardize date format
df['Date'] = pd.to_datetime(df['Date'])

# Calculate boolean attendance (1=P, 0=A)
df['Is_Present'] = df['Status'].apply(lambda x: 1 if x == 'P' else 0)

# Calculate aggregated attendance %
summary = df.groupby('Student_ID').agg(
    Total_Days=('Date', 'nunique'),
    Days_Present=('Is_Present', 'sum')
)
summary['Attendance_Pct'] = (summary['Days_Present'] / summary['Total_Days']) * 100

SQL Insights Layer

Analytical queries for stakeholder reporting

At-Risk Students Pipeline

Identifies bottom quartile students requiring academic intervention.

SELECT 
    s.Student_ID,
    s.Name,
    s.Department,
    a.Attendance_Pct
FROM Student_Dimension s
JOIN Attendance_Fact a ON s.Student_ID = a.Student_ID
WHERE a.Attendance_Pct < 75.0
ORDER BY a.Attendance_Pct ASC;

Departmental Distribution

Aggregates attendance metrics to compare health across disciplines.

SELECT 
    Department,
    COUNT(Student_ID) as Total_Students,
    ROUND(AVG(Attendance_Pct), 2) as Avg_Attendance
FROM Attendance_Summary
GROUP BY Department
HAVING AVG(Attendance_Pct) > 0
ORDER BY Avg_Attendance DESC;
